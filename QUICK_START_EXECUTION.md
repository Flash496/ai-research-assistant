## ğŸš¨ Troubleshooting Quick Links

| Problem | Solution |
|---------|----------|
| Port already in use | Change port in env file or kill process |
| Database won't connect | Check docker-compose.ps and DATABASE_URL in .env |
| Redis connection refused | Start docker: docker-compose up -d redis |
| Module not found | npm install in backend or frontend |
| WebSocket won't connect | Check CORS in app.module.ts |
| Prisma migration failed | npx prisma migrate reset |
| API returns 404 | Check ResearchController is in ResearchModule |
| Frontend can't reach backend | Check NEXT_PUBLIC_API_URL in .env.local |
